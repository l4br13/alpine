# About

alpine linux mini rootfs install script.


# Installation

run install.sh directly without sudo.

when you install on GNU/Linux, it will installed on its repository dir, if you install on Android in Termux app it will installed on home/../alpine

for Android, you can run by typing startalpine

for any linux distros, you can run by
execute the init file inside the repo dir

# Uninstallation

for any linux, you can directly remove the container dir.

for android , just execute uninstall.sh files.
